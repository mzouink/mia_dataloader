Inside __getitem__

1. Select Crop + Location

  • Sampler provides crop index
  • Random 3D origin within crop extent
  • Coordinates in world nm

2. Read Raw EM

  • Convert world coords to voxels
  • Read zarr slice at raw resolution
  • Normalize: (x-min)/(max-min)
  • Zoom to input_size if needed

3. Read Labels

  • Center crop at output_size
  • Per-class zarr read
  • Binarize: val > 0 && val != 255
  • Zoom with nearest-neighbor interp

4. Compute Masks + Transform

  • Spatial mask: crop bounds within output
  • Annotated mask: which classes present
  • Apply EMTransforms (if set)
  • Return 5-tuple

ClassBalancedSampler

Ensures rare organelle classes (e.g., peroxisomes with ~15 crops) are sampled as often as abundant ones (e.g., mito with 200+ crops).

Algorithm

  • Build [n_crops x n_classes] boolean matrix
  • Track running class counts during epoch
  • At each step: pick the least-seen class
  • Randomly sample a crop that annotates it
  • Increment counts for all classes in that crop
  • Repeat for samples_per_epoch steps

Usage

from mia_em_loader import ClassBalancedSampler
from torch.utils.data import DataLoader

sampler = ClassBalancedSampler(
    dataset,
    samples_per_epoch=5000,
    seed=42
)

loader = DataLoader(
    dataset,
    batch_size=4,
    sampler=sampler,
    num_workers=8
)

Transform Pipeline (MONAI)

Spatial Transforms

Applied jointly to raw + labels

  • Rand3DElasticd — elastic deformations
  • RandRotate90d — 90 rotations (3 planes)
  • RandFlipd — flips along each axis

Intensity Transforms

Applied to raw only

  • RandShiftIntensityd — additive shift
  • RandScaleIntensityd — multiplicative scale
  • RandGaussianNoised — Gaussian noise
  • RandGaussianSmoothd — Gaussian blur
  • RandAdjustContrastd — gamma contrast

Default Probabilities

spatial_prob = 0.5
intensity_prob = 0.3
elastic_sigma = (10, 15)
noise_std = (0.01, 0.05)
gamma = (0.7, 1.3)

How to Add Custom Transforms

Transforms are any callable: (raw, labels) -> (raw, labels)

1

Built-in with Custom Params

transforms = get_train_transforms(
    spatial_prob=0.8,
    intensity_prob=0.5,
    noise_std=(0.01, 0.1),
)
2

Config Dict (YAML-friendly)

config = {"spatial_prob": 0.8,
          "intensity_prob": 0.5}
transforms = get_train_transforms_from_config(
    config
)
3

Fully Custom Callable

class MyTransforms:
    def __call__(self, raw, labels):
        # your augmentation logic
        raw = raw + noise
        return raw.clamp(0,1), labels
4

Compose Multiple Stages

pipeline = ComposeTransforms(
    EMTransforms(spatial_prob=0.5),
    MyCustomIntensityAug(),
    RandomCropPaste(),
)

Transform Contract

Your custom transform must follow these rules:

PropertyRequirement
Input raw shape[1, D, H, W] float32, values in [0, 1]
Input labels shape[C, D, H, W] float32, binary (0 or 1)
Output raw shapeSame as input — do not change spatial dims
Output labels shapeSame as input
Raw value rangeKeep in [0, 1] — clamp(0, 1) after augmentation
Labels valuesKeep binary — re-threshold with > 0.5 after interpolation
Spatial consistencyApply spatial transforms to both raw and labels

Example: Wrapping EMTransforms

class MyTransforms:
    def __init__(self):
        self.base = EMTransforms(spatial_prob=0.5, intensity_prob=0.3)

    def __call__(self, raw, labels):
        raw, labels = self.base(raw, labels)     # built-in pipeline
        if torch.rand(1) < 0.2:                  # custom cutout
            raw[:, z:z+s, y:y+s, x:x+s] = 0.0
        return raw.clamp(0, 1), labels

Multi-Dataset Training

ConcatMiaDataset

  • Wraps N sub-datasets into one
  • Validates matching target_classes, input_size, output_size
  • Weighted sampling between datasets
  • Merges crop lists for ClassBalancedSampler
  • Default weights proportional to crop count

Third-Party Datasets

  • Any dataset implementing MiaDataset3D contract works
  • validate_em_dataset() for duck-type checking
  • Plug into same DataLoader + Sampler

Usage

db_a = CropDatabase.from_json("setA.json")
db_b = CropDatabase.from_json("setB.json")

ds_a = CellMapDataset3D(crop_db=db_a, ...)
ds_b = CellMapDataset3D(crop_db=db_b, ...)

combined = ConcatMiaDataset(
    [ds_a, ds_b],
    weights=[0.7, 0.3],
    samples_per_epoch=1000,
)

loader = DataLoader(
    combined,
    sampler=ClassBalancedSampler(combined),
    batch_size=4,
)

Complete Training Example

from mia_em_loader import (
    CropDatabase, CellMapDataset3D, ClassBalancedSampler, get_train_transforms
)
from torch.utils.data import DataLoader

# Load discovered crops
db = CropDatabase.from_json("crops.json")

# Create dataset with augmentations
train_ds = CellMapDataset3D(
    crop_db=db,
    target_classes=["mito", "er", "nucleus"],
    input_size=(128, 128, 128),    # raw patch size
    output_size=(64, 64, 64),      # label patch size (center crop)
    target_resolution=8.0,          # nm per voxel
    samples_per_epoch=5000,
    transforms=get_train_transforms(),
)

# Balanced DataLoader
loader = DataLoader(
    train_ds,
    batch_size=4,
    sampler=ClassBalancedSampler(train_ds, samples_per_epoch=5000),
    num_workers=8,
)

# Training loop
for raw, labels, ann_mask, spatial_mask, meta in loader:
    pred = model(raw)                                       # [B, 1, 128, 128, 128]
    loss = criterion(pred, labels, ann_mask, spatial_mask)   # [B, 3, 64, 64, 64]
    loss.backward()
    optimizer.step()
